### 기본적인 제어
<a id="기본적인-제어"></a>

#### cd (change directory)

지정한 디렉토리로 이동
```shell
cd /home/user/Documents //직접지정
cd ~ //홈 디렉토리
cd .. // 상위 디렉토리
cd - // 이전 디렉토리
```

#### pwd (print working directory)

현재 작업 중인 디렉토리(폴더)의 절대 경로를 출력

```shell
pwd
```

#### ls (list)

디렉토리의 내용물을 나열하는 명령어로   
파일, 디렉토리, 링크 등을 보여주며, 선택적으로 다양한 정보를 함께 표시할 수 있다

```shell
ls //기본값은 현제 디렉토리에서의 내용물 출력
ls -l //파일 타입, 권한, 링크 수, 소유자, 그룹, 크기, 타임스탬프, 파일 이름 등이 포함
ls -a //숨김 파일을 포함한 모든 파일을 보여줌
ls -r결과를 역순으로 출력
```

#### mv (move)

파일이나 디렉토리를 이동하거나 이름을 변경하는 데 사용

```shell
mv [옵션] 원본 대상
mv file.txt /home/user/documents //이동하는 예시
mv file.txt newfile.txt //이름을 변경하는 예시
```

mv 명령어는 기존의 대상 파일을 덮어쓸 수 있으므로 주의해야 합니다.   
파일을 이동하거나 이름을 변경하기 전에 항상 주의하고, 필요한 경우 백업을 만들어 두는 것이 좋습니다.

mv 명령어의 옵션

- `-i`: 파일 덮어쓰기 전에 사용자에게 확인을 요청합니다.
- `-u`: 대상 파일이 원본 파일보다 더 오래된 경우에만 원본 파일을 이동하거나 이름을 변경합니다.
- `-v`: 파일 이동이나 이름 변경 정보를 자세히 출력합니다.

#### cp (copy)

파일 또는 디렉토리를 복사하는데 사용되는 명령어

```shell
cp myfile.txt myfile_copy.txt //파일복사
cp -r mydir mydir_copy // 디텍토리복다
```

#### mkdir (make directory)

새로운 디렉토리(폴더)를 생성하는 명령어

```shell
mkdir testdir // 1개 생성
mkdir dir1 dir2 // 여러개 생성
```

#### rm (remove)

파일이나 디렉터리를 삭제하는 명령어

```shell
rm test.txt //1개 삭제
rm file1.txt file2.txt file3.txt //2개 삭제
rm -r my_directory //디렉터리와 그 내부의 모든 파일 및 서브 디렉터리 삭제
```

> rmdir (remove directory) 도 있지만 rm하나도 파일까지 지울 수 있으면 이게 좋은 듯 합니다.
> 

#### cat (concatenate)

파일의 내용을 표준 출력(stdout)으로 보내거나 파일을 연결하거나 생성하는 데 사용

```shell
cat filename.txt //파일의 내용 출력
cat file1.txt file2.txt > mergedfile.txt //여러 파일 합치고 > 여기에 저장
cat > newfile.txt //새 파일 생성
cat file1.txt > file2.txt //파일의 내용을 다른 파일로 복사(이미 존재한다면 덮어쓰기)
```



### 텍스트 에디터(vim)
<a id="vim"></a>

"vi"와 "vim"은 모두 텍스트 에디터로, 유닉스와 유닉스 기반 시스템에서 널리 사용됨.

#### vim 모드

- Command Mode(명령 모드)
  - 텍스트 편집이 아닌 커서 이동, 복사 및 붙여넣기, 삭제 등의 명령을 수행할 수 있음
  - vi 에디터를 처음 시작하면 기본적으로 이 모드로 시작
- Insert Mode(입력 모드)
  - 실제로 텍스트를 입력하거나 수정할 수 있음.
  - 명령 모드에서 `i`, `a`, `o` 등의 키를 눌러 입력 모드로 전환할 수 있음
- Line Command Mode(라인 명령 모드)
  - 파일 저장, vi 종료, 검색 및 치환 등의 명령을 수행할 수 있음
  - 명령 모드에서 `:` 키를 눌러 라인 명령 모드로 전환할 수 있음

#### 편집

vim 에디터에서 Insert Mode(입력모드)로 전환하는 방법

1. `i`: 현재 커서 위치에서 인서트 모드로 전환
2. `I`: 현재 줄의 처음으로 이동하고 인서트 모드로 전환
3. `a`: 현재 커서 위치의 다음 문자 위치에서 인서트 모드로 전환
4. `A`: 현재 줄의 끝으로 이동하고 인서트 모드로 전환
5. `o`: 현재 줄 아래에 새로운 줄을 만들고 그 위치에서 인서트 모드로 전환
6. `O`: 현재 줄 위에 새로운 줄을 만들고 그 위치에서 인서트 모드로 전환

#### 이동

커서를 이동

1. `0` (zero): 현재 줄의 시작으로 이동
2. `$`: 현재 줄의 끝으로 이동
3. `G`: 문서의 마지막 줄로 이동
4. `gg`: 문서의 첫 줄로 이동
5. `[숫자]G`: 지정한 줄 번호로 이동합니다. 예를 들어, `10G`는 10번째 줄로 이동
6. `%`: 현재 줄에서 괄호, 중괄호, 대괄호 등의 짝을 찾아 이동

#### 문자 삭제

1. `x`: 현재 커서 위치의 문자를 삭제
2. `dw`: 현재 커서 위치에서 단어의 끝까지 삭제
3. `d$`: 현재 커서 위치에서 줄의 끝까지 삭제

#### 행 삭제

1. `dd`: 현재 줄을 삭제
2. `[숫자]dd`: 지정한 수만큼의 줄을 삭제. 예를 들어, `2dd`는 현재 줄을 포함해 아래로 2줄을 삭제

#### 복사

1. `yy` 또는 `Y`: 현재 줄을 복사
2. `[숫자]yy` 또는 `[숫자]Y`: 지정한 수만큼의 줄을 복사. 예를 들어, `2yy` 또는 `2Y`는 현재 줄을 포함해 아래로 2줄을 복사
3. `yw`: 현재 커서 위치에서 단어의 끝까지 복사합니다.

#### 붙여넣기

1. `p`: 현재 커서 위치의 다음 줄에 복사한 또는 삭제한 텍스트 붙여넣기
2. `P`: 현재 커서 위치의 이전 줄에 복사한 또는 삭제한 텍스트 붙여넣기

#### 찾기와 치환

1. `/[찾을 문자열]`
  - 문서에서 앞으로 찾을 문자열을 검색
  - 예를 들어, `/hello`는 문서에서 "hello"를 앞으로 검색
2. `?[찾을 문자열]`
  - 문서에서 뒤로 찾을 문자열을 검색
  - 예를 들어, `?hello`는 문서에서 "hello"를 뒤로 검색
3. `n`: 이전에 검색한 방향으로 다음 검색 결과로 이동
4. `N`: 이전에 검색한 반대 방향으로 다음 검색 결과로 이동
5. `:[범위]s/[찾을 문자열]/[바꿀 문자열]/[플래그]`
  - 범위 내에서 찾을 문자열을 바꿀 문자열로 치환
  - 범위는 생략하면 현재 줄이 되며, `%`를 사용하면 전체 문서가 됨
  - 플래그는 생략하면 첫 번째 일치하는 문자열만 바꾸며, `g`를 사용하면 모든 일치하는 문자열을 변경함
  - 예를 들어, `:%s/hello/world/g`는 문서 전체에서 "hello"를 "world"로 바꿉니다

#### 저장과 종료

1. `esc` 키를 눌러 편집 모드에서 명령 모드로 전환
2. `:w`: 현재 파일을 저장
3. `:q`: Vim 에디터를 종료 (단, 변경사항이 저장되지 않았다면 에러 메시지가 출력됨)
4. `:q!`: 변경사항을 저장하지 않고 Vim 에디터를 강제로 종료
5. `:wq`: 현재 파일을 저장하고 Vim 에디터를 종료합
6. `ZZ`: 현재 파일을 저장하고 Vim 에디터를 종료 (`:wq`와 동일한 기능)
7. `:x`: 현재 파일을 저장하고 Vim 에디터를 종료 (`:wq`와 동일한 기능)
