## 💡 문자열
<a id="String"></a>

"" 큰 따옴표를 사용하여 문자로 구성된 문장을 문자열이라고 함.
```java
"123"
"Hello World!"
"s"
// 한글자, 숫자 도 ""큰따옴표 안에 있으면 문자열에 포함
```
